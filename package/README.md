# chat-app-types

chat-app全体で利用する型情報を定義するパッケージ

> [!NOTE]
> 主に、APIコールに必要な引数の型やAPIのレスポンスの値の型等を定義する

## パッケージのインストール

`npm i @kazeusagi/chat-app-types`

## パッケージの更新

1. パッケージの内容を更新する
1. `package.json` の `version` を更新する
1. `npm login`
1. `npm publish`
